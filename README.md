# 🚀 Agency2 AI

An AI-powered modern agency website built with React, Vite, and Tailwind CSS. This project showcases a professional digital agency platform with a modern UI, responsive design, smooth user experience, and AI-focused branding.

🌐 Live Demo: https://agency2-ai-ankit-git-main-ankitpandey987359-stacks-projects.vercel.app/

---

## ✨ Features

- 🎨 Modern and responsive UI
- 🌙 Dark mode support
- ⚡ Fast performance with Vite
- 📱 Mobile-friendly design
- 🤖 AI agency themed website
- 📞 Contact section
- 🛠 Services showcase
- 👨‍💻 Clean component-based architecture
- 🚀 Deployed on Vercel

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/agency2-ai.git
```

Move into project directory

```bash
cd agency2-ai
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
